## 1. Выполните следующие замены

### 1.1. `a -> @`

Команда:
```
$ sed 's/a/@/g' google-10000-english.txt 
```

### 1.2. `er в конце слова -> x0r`

Команда:
```
$ sed 's/er\>/x0r/g' google-10000-english.txt 
```

### 1.3. `and -> &`

Команда:
```
$ sed 's/@nd/\&/g' google-10000-english.txt 
```
